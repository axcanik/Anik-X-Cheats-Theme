## 📥 Installation Guide

Installing the **ANIK X CHEATS** theme on your server is incredibly easy. We have provided an automated bash script to do all the heavy lifting for you!

### Prerequisites
* A server running **Ubuntu 20.04 / 22.04**.
* Pterodactyl Panel (v1.x) already installed and fully working.
* Root access to your server.

### ⚡ Automated Installation (Recommended)

Simply SSH into your VPS as `root` and run the following command to download and execute our installer:

```bash
wget -qO install.sh https://raw.githubusercontent.com/axcanik/Anik-X-Cheats-Theme/main/install.sh && bash install.sh
```

**What the script does:**
1. Backs up your existing panel files.
2. Downloads the latest `gamenet_theme_v2.zip` release.
3. Injects our premium CSS and React components.
4. Rebuilds the frontend panel automatically.
5. Clears Laravel caches to instantly apply the new look.
